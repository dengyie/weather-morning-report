# Weather Morning Report

OpenPet plugin package for action-oriented weather reports.

## Commands

- `refresh`: fetch weather and generate a report
- `announce`: fetch weather and speak the report
- `last`: speak the last cached report
- `status`: show cache and config status
- `clear-cache`: clear plugin storage

## Permissions

- `network`
- `pet:say`
- `storage`

## Network

- `wttr.in`
- `wttr.is`
